Projects

Sasha Game Engine Project

The "Sasha" game engine project showcases my dedication to mastering C++ and delving into the intricacies of game development. Inspired and guided by Cherno's comprehensive YouTube tutorials, this project represents more than just an academic exercise. It is my commitment to applying advanced programming concepts in a tangible, engaging manner. The choice to do this project was driven by a desire to challenge myself, and to translate theoretical learning into a practical toolkit for my different creative expressions.

This engine incorporates modern rendering techniques, leveraging OpenGL for graphics rendering to support both 2D and 3D visuals, and implements shader programming to achieve dynamic lighting and shadow effects, enhancing the visual fidelity of game environments. Its architecture is a custom entity-component system (ECS), facilitating a modular approach to game object management and scene composition, optimizing performance and scalability. The engine also has a physics engine for realistic collision detection and response, integrated with an efficient memory management system to ensure optimal resource utilization. Also, it includes a versatile input handling system to process player commands across various input devices, alongside an audio system for sound effects and music management.

The "Sasha" game engine not only exhibits my skills in C++ programming but also my ability to navigate and piece together the many aspects of game development, from system architecture to real-time rendering algorithms.

Black-Spot Tank Robot Project

I created the "Black-Spot" Tank Robot project because I'm really interested in how software and hardware work together to make real-world things. I wanted to create something that can be used by people to interact with one another from across the world. I used a Raspberry Pi and Arduino to push my skills in programming and engineering and to show how Internet of Things (IoT) devices can make robotics fun and more accessible.

The inclusion of the Black-Spot Tank Robot in my portfolio shows a broad spectrum of my capabilities, spanning both hardware and software development. The development of the Black-Spot Tank Robot required a deep dive into both the mechanical and computational realms, which integrated various technologies, such as controlling a Arduino using a Raspberry Pi. From designing the physical chassis to programming the control logic, every step was a lesson in hardware and software design.

Critical thinking and project management skills were crucial throughout the development process, especially when integrating a camera for enhanced user interaction. This addition not only improved the robot's interactivity but also introduced the complexity of real-time video processing. Similarly, implementing remote control capabilities demanded a thorough understanding of network communications, ensuring the robot could be maneuvered effectively over the internet.
