Projects

Sasha Game Engine Project

I have always been interested in game design. After gaining academic experience with several programming concepts and languages I wanted to put them to the test, so I set out to create a game engine. Although I have no formal knowledge of game engine design, I created the “Sasha” game engine with the help of Cherno’s YouTube tutorials to challenge myself and to gain a better understanding of the C++ language. I like to name my projects and I named it “Sasha” because I had a cat at the time that tragically ran away, and it seemed like a fitting name for this engine to remember her by.

I was driven by the challenge of applying theoretical programming knowledge to something that practical. This game engine is designed with many modern features for rendering techniques. These include OpenGL for both 2D and 3D graphics that incorporates shader programming for dynamic lighting and shadow effects. It has a physics and memory management system for realistic collision detection and response. Input handling implementation allows for users to use a variety of controls and input devices. It also features a custom entity-component system (ECS) that utilizes a modular approach for efficient game object management and scene composition. This greatly optimizes performance and scalability in the engine.

For future implementation I hope to integrate networking capabilities and cross-platform compatibility. Also, I plan to improve the engines handling of game assets to make faster load times and performance during gameplay.

This game engine is meant to showcase my proficiency in C++ programming concepts and my ability apply them in a real-world application.

Black-Spot Tank Robot Project

I created the "Black-Spot" Tank Robot project because I'm really interested in how software and hardware work together to make real-world things. I wanted to create something that can be used by people to interact with one another from across the world. I used a Raspberry Pi and Arduino to push my skills in programming and engineering and to show how Internet of Things (IoT) devices can make robotics fun and more accessible.

The inclusion of the Black-Spot Tank Robot in my portfolio shows a broad spectrum of my capabilities, spanning both hardware and software development. The development of the Black-Spot Tank Robot required a deep dive into both the mechanical and computational realms, which integrated various technologies, such as controlling a Arduino using a Raspberry Pi. From designing the physical chassis to programming the control logic, every step was a lesson in hardware and software design.

Critical thinking and project management skills were crucial throughout the development process, especially when integrating a camera for enhanced user interaction. This addition not only improved the robot's interactivity but also introduced the complexity of real-time video processing. Similarly, implementing remote control capabilities demanded a thorough understanding of network communications, ensuring the robot could be maneuvered effectively over the internet.
